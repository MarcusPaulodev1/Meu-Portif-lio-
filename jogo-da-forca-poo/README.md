# 🎮 Jogo da Forca em Python (POO)

Projeto desenvolvido em Python utilizando os princípios de Programação Orientada a Objetos (POO).

## 📚 Objetivo do Projeto

O objetivo deste projeto foi desenvolver uma versão completa do jogo da forca utilizando:

- Classes
- Objetos
- Encapsulamento
- Separação de responsabilidades
- Organização modular do código

Além de fazer o jogo funcionar, o foco principal foi criar uma estrutura limpa, organizada e fácil de manter.

---

# 🧠 Estrutura do Projeto

O sistema foi dividido em 3 classes principais:

## 🔹 Classe Boneco
Responsável pelo controle visual da forca.

### Funções:
- controlar erros
- definir dificuldade
- avançar o boneco
- verificar derrota
- mostrar o desenho atual

---

## 🔹 Classe PalavraSecreta
Responsável pela lógica da palavra.

### Funções:
- armazenar a palavra
- verificar letras
- revelar letras corretas
- mostrar progresso da palavra
- verificar vitória

---

## 🔹 Classe JogoDaForca
Classe principal do sistema.

### Funções:
- iniciar a partida
- controlar fluxo do jogo
- receber entradas do usuário
- validar letras
- atualizar tela
- coordenar as outras classes

# 📌 Dificuldades do Jogo

## Fácil
- Mais chances de erro
- Progressão mais leve

## Normal
- Funcionamento tradicional

## Difícil
- Cada erro possui peso maior
- Derrota mais rápida
