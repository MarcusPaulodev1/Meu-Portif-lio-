# 🎲 RPG Simples Aleatório

Esse projeto nasceu como um trabalho da faculdade, mas acabou ficando interessante o suficiente para entrar no meu portfólio.  
A ideia é simples: criar um pequeno RPG em Python usando programação orientada a objetos, com personagens, missões, inimigos e batalhas aleatórias.

Não é um jogo gigante, nem tenta ser um RPG completo com mapa, inventário complexo ou sistema gráfico. A proposta aqui foi manter tudo simples, direto e funcional, mas com uma base legal para evoluir depois.

## 🧙 O que o jogo faz?

- Cria um jogador
- Cria um personagem com classe aleatória
- Gera atributos como ataque e defesa
- Permite escolher missões
- Gera inimigos aleatórios
- Executa batalhas por turno
- Dá experiência ao vencer missões
- Possui sistema simples de nível
- Recupera um pouco da vida após vitórias

## 🛠️ Tecnologias usadas

- Python 3
- Programação Orientada a Objetos
- Biblioteca `random`

## ▶️ Como executar

No terminal, rode:

```bash
python rpg_simples.py
```

## 📌 Objetivo do projeto

O objetivo principal foi praticar conceitos básicos de Python e POO, como:

- Classes
- Objetos
- Métodos
- Atributos
- Listas
- Condicionais
- Laços de repetição
- Aleatoriedade
